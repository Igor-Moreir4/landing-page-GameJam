Dúvida frequente

O que é uma GameJam?

Kratos

"Uma Game Jam é um evento onde desenvolvedores de jogos se reúnem para criar jogos em um período de tempo limitado. Durante a Game Jam, os participantes trabalham em equipes ou individualmente para criar um jogo a partir do zero, seguindo um tema específico ou desafio proposto pelos organizadores"


Mario

" Esses eventos são ótimas oportunidades para os desenvolvedores exercitarem sua criatividade, aprenderem novas habilidades . No final da Game Jam, os jogos são geralmente apresentados e avaliados, muitas vezes com prêmios para os melhores projetos."
